Project Ft_ls
 -------------------------------------------------------------------------------
The executable should work in the same manner as the system ls
with the following flags
-l (long format)
-a (include dir's that start with '.')
-r Reverse the order of the sort to get reverse lexicographical order
or the oldest entries first (or largest files last, if combined with sort by size
-R (search dir's recursively)
-t (sort by time modified before sorting lexicographically)
